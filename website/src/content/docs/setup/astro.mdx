---
title: Setup Spotlight for Astro
description: A guide how to setup Spotlight for Astro
---

To install Spotlight for Astro call:

import { Tabs, TabItem } from '@astrojs/starlight/components';

<Tabs>
	<TabItem label="npm">
		```sh 
		npx astro add @sentry/spotlight
		```
	</TabItem>
	<TabItem label="pnpm">
		```sh 
		pnpm astro add @sentry/spotlight
		```
	</TabItem>
	<TabItem label="yarn">
		```sh 
		yarn astro add @sentry/spotlight
		```
	</TabItem>
</Tabs>
