title: Run Sidecar with NPX

import { Tabs, TabItem } from '@astrojs/starlight/components';

To run Sidecar independently with NPX, run the following command:

<Tabs>
  <TabItem label="npm and yarn">
    ```sh
    npx @spotlightjs/spotlight [optionalPort]
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```sh
    pnpx @spotlightjs/spotlight [optionalPort]
    ```
  </TabItem>
</Tabs>


Replace `[optionalPort]` with the desired port number. If no port number is provided, the default port is set to `8969`.

### Example

<Tabs>
  <TabItem label="npm and yarn">
    ```sh
    npx @spotlightjs/spotlight 3001
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```sh
    pnpx @spotlightjs/spotlight 3001
    ```
  </TabItem>
</Tabs>

This command will start the Sidecar server on port `3001`. If no port is specified, the server will default to port `8969`.
